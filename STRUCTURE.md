# Pythonic-Design-Patterns Structure

This repository backs `Pythonic Design Patterns`. Active generated projects use
`README.md`, `starter/`, and `solution/`.

Folders not linked by the live catalog on 2026-05-14 were moved to
`_archived-unlinked/`. Those archived folders are historical or promotable
material, not active public course projects.

New project folders should use complete deterministic `PDP-*` slugs and should
be linked from the live catalog in the same change.
